# Changelog

All notable changes to Vi-Player will be documented here.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

This project follows Semantic Versioning:

- **Major** (`0.x.0`) — New features and architectural changes.
- **Minor** (`0.0.x`) — Bug fixes, optimizations and small improvements.

## [Unreleased]

### Added

- UTF-8 support
- Multiple `:set` options parsing
- `H` and `L` keys for beggining/end playback navigation

### Fixed

- Incorrect home screen version text
- StatusLine widget text errors

## v0.1.1 - 2027-07-04

### Fixed

- Command parsing edge cases
- Minor UI inconsistences

### Changed

- Improved default colorscheme
- Improved default colors and highlights

## v0.1.0 - 2027-06-30

### Added

- Vim-like navigation
- Commandline mode
- playlist loading
- MPV playback
- Colorscheme support
- Installable package structure
- Automatic config bootstrap
